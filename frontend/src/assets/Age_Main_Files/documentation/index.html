The template on the purpose of making newspaper/magazine site. This will help you publish your news media.

Dependency

The template made with LESS preprocessor to generate CSS. And some jQuery plugin to give a nice look and functionality. We use Google Font to make the template. You must have to keep these jquery plugins if you wish the template work same to you as preview. 

 

The template created with the help of: 

  • Twitter Bootstrap 3x
  • Google Font
  • LESS CSS
  • Jquery
